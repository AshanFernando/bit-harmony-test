## Getting started

Run the following commands:

1. `yarn add global @teambit/bit`

2. `bbit login`

3. `git clone git@github.com:teambit/getting-started-result.git getting-started`

4. `cd getting-started`

5. `bbit install`

6. `bbit start`

Explore the workspace at http://localhost:3000/
